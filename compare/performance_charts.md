### 5. 性能分析总结

**VerseConf vs TOML**:
- VerseConf 解析速度比 TOML **快 2.8-4.1 倍**
- 小文件时优势更明显（4.1x），大文件时稳定在 2.8x
- TOML 解析器在处理复杂嵌套结构时性能下降显著

**VerseConf vs JSON**:
- JSON 解析速度比 VerseConf **快 2-10 倍**
- 小文件时 JSON 优势明显（10x），大文件时差距缩小至 2x
- `serde_json` 库经过高度优化，是 Rust 生态中最快的 JSON 解析器

**文件大小**:
- VCF 和 TOML 大小相近（VCF 约大 8-12%）
- JSON 在小文件时最小（无注释、无多余符号）
- 大文件时 VCF 比 JSON 小约 13-14%

**关键发现**:
1. VerseConf 在解析性能上显著优于 TOML
2. JSON 在解析速度上仍然领先，但 VerseConf 差距在可接受范围内
3. VerseConf 支持增量解析和缓存，在长期运行场景中有巨大优势
4. 文件大小方面，VCF 和 TOML 相当，都比 JSON 更紧凑（大文件场景）
